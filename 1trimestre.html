Sistemas de Numeração

Esse conteúdo foi algo novo para mim, sendo que durante o ensino fundamental aprendi somente sobre o sistema decimal. Achei o conteúdo interessante, no início tinha alguma dificuldade por se tratar de algo novo, mas após um tempo consegui entender com tranquilidade.


É um conjunto de diferentes sistemas que são compostos por símbolos que representam quantidades numéricas. De acordo com a quantidade de símbolos é denominada a base do sistema.

EX:

Um sistema que possui dez símbolos é chamado de Sistema Decimal ou Sistema de base dez.


Os quatro Sistemas de Numeração que estudamos durante o trimestre são:

  • Sistema Decimal

  • Sistema Binário

  • Sistema Octal

  • Sistema Hexadecimal
